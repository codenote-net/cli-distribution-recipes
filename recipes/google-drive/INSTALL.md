# Install codenote-hello

This archive contains the npm package tarball for `@codenote-net/hello-cli`.

## Requirements

- Node.js 22 or newer
- npm

## Install

From the directory where you extracted this zip, run:

```sh
npm install -g ./codenote-net-hello-cli-0.1.0.tgz
```

## Verify

Run:

```sh
codenote-hello
```

Expected output:

```text
Ohayou gozaimasu, Konnichiwa, Konbanwa!
```

## Uninstall

```sh
npm uninstall -g @codenote-net/hello-cli
```
